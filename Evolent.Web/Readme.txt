Evolent Web MVC Project
-------------------------------
This MVC Web api sample is created to use Evolent Web Api and perform CRUD Operations on database using Evolent Api 

If Api port change then need to modify below key in Web.Config 
 <add key="EvolentServiceApiUrl" value="http://localhost:60252/api/"/>

Above project is created using Visual Studio 2015 Web Express and MVC5 and 
Service layer using Interface.
